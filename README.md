****How to remove Branches ****
1-Locally:
git branch -d branch_name


2-Remotely:
git origin -d branch_name

*****list tags****
git tag -l

*** Delete Tags ******
**Locally:
git tag -d tag_name

**Remotely:
git push --delete origin tag_name
